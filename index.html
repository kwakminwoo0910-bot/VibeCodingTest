명지대 바이브 코딩 부트캠프

AI와 함께하는 개발의 새로운 패러다임을 제시합니다. 프롬프트 엔지니어링부터 UI/UX, 백엔드, 배포까지 실무 중심의 최신 기술을 경험하세요.

부트캠프 소개

명지대 바이브 코딩 부트캠프는 단순한 코딩 교육을 넘어, AI를 활용한 창의적인 문제 해결 능력을 기르는 과정입니다. 최신 AI 도구와 기술 스택을 활용하여 아이디어를 현실로 만드는 모든 과정을 직접 경험하고, 자신만의 프로젝트를 완성하게 됩니다.

무엇을 배우나요?

1일차

바이브 코딩의 이해와 환경세팅, 기반지식 학습, 다양한 분야에 활용, 배포, 바이브 코딩의 프롬프트 입력 방법, CursorAI, ChatGPT를 활용하여 주제에 맞는 파이썬 코드 작성 실습

2일차

바이브 코딩을 위한 디자인, UI/UX 디자인, Firebase 구조 소개 및 설정, Firebase와 FastAPI 연동 실습

3일차

백엔드 연결 등 고급 기능 설계, 고급 기능을 위해 알아야 할 지식, 프롬프트 기반 발성법, 문제 정의 및 페르소나 설정, React 기본 구조 및 Vite 환경 설정

4일차

Firebase 연동 로그인 UI 구성, FastAPI 기본 구조, Firebase 인증 API 및 CRUD 실습

5일차

MCP 개요 및 Agent 설계, 랭크파와 MCP 연동 실습, GitHub 기반 협업

6일차

TDD, 코드 리뷰, 커밋 메시지, 문서화 실습, Firebase Hosting, GCP 기본, 배포 실습 및 디버깅, 문제 해결 제안서 작성 워크숍
